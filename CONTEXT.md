# Typing Practice

A focused reading experience in which people practice typing longer-form text and request help understanding unfamiliar words without leaving the session.

## Language

**Typing Session**:
An open-ended activity in which a user progresses through a Reading Source by typing until they stop or reach its end. It has no timer or speed goal.
_Avoid_: Test, game, race, reading mode

**Reading Source**:
A finite, redistribution-authorized English-language book or article selected for typing and reading.
_Avoid_: Test text, prompt, exercise

**Catalog**:
The private, curated collection of Reading Sources published for the sole reader. The reader selects sources from the Catalog but does not add or edit them through the site.
_Avoid_: User library, uploads

**Import Draft**:
A proposed Reading Source extracted from a URL, normalized only with the approved typing-friendly character replacements, and awaiting operator review. AI annotations may accompany it, but it is not part of the Catalog until explicitly approved.
_Avoid_: Upload, published source

**Publish**:
The operator's explicit approval that an Import Draft is accurate and authorized for redistribution, adding it to the Catalog.
_Avoid_: Import, save, deploy

**Reading Progress**:
The point a reader has reached in a Reading Source across one or more Typing Sessions on the same device.
_Avoid_: Score, result, words per minute

**Word Help**:
An on-demand Traditional Chinese explanation and English usage example for a CEFR B2+ term, idiom, or contextually unusual meaning identified while importing a Reading Source.
_Avoid_: Dictionary popup, definition panel

**Typing Error**:
An incorrect character that remains highlighted after Reading Progress advances. It provides immediate feedback but is not scored; the reader can use Backspace to return and correct it.
_Avoid_: Failure, penalty, accuracy result
